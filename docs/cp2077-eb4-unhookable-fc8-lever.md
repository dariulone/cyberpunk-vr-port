(1st deploy survived gameplay by luck, crashed on inventory; 2nd deploy crashed on F7.) Disabled permanently.
- **E28 (sub_140167E28) is OFF the gameplay path**: hooking it never fired (0 callbacks) - gameplay reaches EB4 via job-table dispatch, not E28's direct call. So E28 wrapper is used only in some non-gameplay context.

Preserves all non-vols + eye-1 rax, ABI-clean (sub rsp,0x38 per call). F12 done-log measures builds via scene-pass work delta (g_sceneDrvCallTotal): DOUBLED vs NOT doubled. Test: F7 (expect `FGB call-site hook result: SUCCESS`) in gameplay, F12, watch crash/artifacts, send log. If DOUBLED + no crash = first working simultaneous seam.
